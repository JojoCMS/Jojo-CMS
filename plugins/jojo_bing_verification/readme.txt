Bing Webmaster Center requires verification that you are in charge of the domain name. This is achieved by uploading an XML file onto your web server that contains the user codes of verified users.

This plugin makes this process a little easier. Instead of uploading a file and user details via FTP, you can simply copy-paste the user details into the "Edit Options" part of Jojo CMS.

Our goal is to keep the root folder clean and tidy as much as possible, and this is another way of doing this.

Multiple valid usernames can be entered - simply add one per line.
